# ASL Recognition: MediaPipe + Random Forest

## Основная идея

Вместо использования сложных нейросетей для обработки изображений, этот подход использует:
1. **MediaPipe** для детекции 21 ключевой точки руки
2. **Нормализацию координат** относительно запястья
3. **Random Forest** для классификации по 63 числовым признакам

##  Ключевые преимущества

| Метрика | MediaPipe + Random Forest | Нейросеть (YOLO + ConvNeXt) |
|---------|---------------------------|----------------------------|
| **Время обучения** |  2 минуты |  6 часов |
| **Размер модели** |  2 MB |  30+ MB |
| **Скорость (FPS на CPU)** |  90-100 |  15-20 |
| **Точность** |  97% |  96% |

##  Технологический стек
